Scenario Comprar productes scenario

NameComprar productes scenario
DescriptionQuan l’usuari, després d’haver seleccionat els productes que vol comprar, fa una petició de compra, se li demanen les seves preferències d’enviament i dades de pagament. Quan es validen aquestes dades, el sistema verifica la disponibilitat i valida la transacció. Dins d’aquest escenari hi ha dos subescenaris: - Enviament de productes: Quan s’ha validat la transacció d’una compra, el sistema prepara les dades d’enviament. Després d’un procés de negociació d’enviament, el sistema informa l’usuari de l’estat de l’enviament i proporciona un número de seguiment i les dades del transportista. El transportista recull els productes i actualitza l’estat del paquet. Quan l’usuari rep el producte, confirma la recepció. - Pagament de productes: Quan s’ha validat la transacció de l’enviament d’una compra, el sistema redirigeix l’usuari a la passarel·la de pagament i aquest introdueix les dades de pagament. Es processa la transacció i el sistema registra l’estat del pagament i informa l’usuari amb la factura, en cas que els productes comprats provinguin d’una botiga externa, es processa el pagament amb aquesta.

Scenario Retornar productes scenario

NameRetornar productes scenario
DescriptionQuan l’usuari sol·licita la devolució d’un producte comprat i el sistema verifica l’estat i la política de devolució de la botiga, aquesta accepta o rebutja la sol·licitud i s’informa l’usuari.

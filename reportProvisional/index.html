Scenario Comprar productes scenario

NameComprar productes scenario
DescriptionQuan l’usuari, després d’haver seleccionat els productes que vol comprar, fa una petició de compra, se li demanen les seves preferències d’enviament i dades de pagament. Quan es validen aquestes dades, el sistema verifica la disponibilitat i valida la transacció. Dins d’aquest escenari hi ha dos subescenaris: Enviament de productes Quan s’ha validat la transacció d’una compra, el sistema prepara les dades d’enviament i genera l’ordre de transport. El sistema informa l’usuari de l’estat de l’enviament i proporciona un número de seguiment i les dades del transportista. El transportista recull els productes i actualitza l’estat del paquet. Quan l’usuari rep el producte, confirma la recepció. Pagament de productes Quan s’ha validat la transacció de l’enviament d’una compra, el sistema redirigeix l’usuari a la passarel·la de pagament i aquest introdueix les dades de pagament. Es processa la transacció i el sistema registra l’estat del pagament i informa l’usuari amb la factura.

Scenario Buscar productes scenario

NameBuscar productes scenario
DescriptionQuan l’usuari fa la petició de buscar un o varis productes al sistema introduint una sèrie de paràmetres, el sistema processa la consulta i li retorna tota la informació detallada dels productes trobats. L’usuari revisa els productes i pot seleccionar un per obtenir més informació o afegir-lo a la compra.

Scenario Retornar productes scenario

NameRetornar productes scenario
DescriptionQuan l’usuari sol·licita la devolució d’un producte comprat i el sistema verifica l’estat i la política de devolució de la botiga, aquesta accepta o rebutja la sol·licitud i s’informa l’usuari. Si és acceptada, el sistema gestiona la logística de retorn, l’usuari envia el producte i el sistema confirma la recepció i processa el reemborsament.

NameGestionar Feedback scenario
